🎯 O que são HTML5 e CSS?

HTML5 (HyperText Markup Language)

HTML é a estrutura de uma página web. Pense nele como o esqueleto de uma casa - ele define onde fica cada coisa: títulos, parágrafos, imagens, links, etc.

CSS (Cascading Style Sheets)

CSS é o estilo visual da página. É como a pintura e decoração da casa - cores, tamanhos, posicionamento, animações, etc.

💡 Dica: HTML e CSS trabalham juntos! HTML cria a estrutura e CSS deixa tudo bonito e organizado.

🛠️ Estrutura Básica de um Documento HTML

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Esta é minha primeira página web.</p>
</body>
</html>

Explicação de cada parte:

  • <!DOCTYPE html> - Diz ao navegador que é um documento HTML5
  • <html> - Tag principal que envolve tudo
  • <head> - Informações sobre a página (não aparece na tela)
  • <body> - Conteúdo visível da página

🎨 Como Adicionar CSS

1. CSS Inline (dentro da tag)

<p style="color: blue; font-size: 20px;">Texto azul</p>

2. CSS Interno (na tag <style>)

<style>
    p {
        color: blue;
        font-size: 20px;
    }
</style>

3. CSS Externo (arquivo separado) - RECOMENDADO

<link rel="stylesheet" href="estilo.css">
💡 Melhor Prática: Use CSS externo! Deixa o código organizado e você pode reutilizar o mesmo estilo em várias páginas.

🎯 Seletores CSS

Seletor de Tag

p {
    color: red;
}

Seleciona TODAS as tags <p>

Seletor de Classe (.)

.minha-classe {
    color: blue;
}

Usa: <p class="minha-classe">

Seletor de ID (#)

#meu-id {
    color: green;
}

Usa: <p id="meu-id">

💡 Importante:
  • Classes podem ser usadas várias vezes
  • IDs devem ser únicos na página

📐 Flexbox - Layout Flexível

Flexbox facilita MUITO o alinhamento de elementos!

.container {
    display: flex;
    justify-content: center;    /* Alinha horizontalmente */
    align-items: center;        /* Alinha verticalmente */
    gap: 20px;                  /* Espaço entre itens */
}

🎯 Grid - Layout em Grade

Grid CSS é ideal para layouts mais complexos!

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;  /* 3 colunas iguais */
    gap: 20px;                            /* Espaço entre */
}
