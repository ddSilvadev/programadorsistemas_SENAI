Aprendemos sobre algumas tags HTML, como fazer CSS interno e CSS inline


O HTML cria a estrutura, mas o visual é pobre e rigido ai que entra o CSS


Estrutura basica do CSS e tipos de CSS::

na pagina styles.css, dedicada somente para o CSS permite a criação e mantimento de centenas de items. Opção mais profissional para desenvolvimento

.selector/classe {
    propriedade: valor;
    color: #ffffff;
}



inline, na propria tag. Conveniente porém mais dificil de manter

CSS interno, no topo da pagina

    <style>
