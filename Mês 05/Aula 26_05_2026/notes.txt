INICIO MÓDULO BANCO DE DADOS


O SQL (Structured Query Language) foi criado na década de 1970 pela IBM, baseado no modelo relacional desenvolvido por Edgar F. Codd. Seu objetivo era facilitar o armazenamento, organização e consulta de dados em bancos de dados relacionais. Com o tempo, o SQL se tornou o padrão mundial para gerenciamento de bancos de dados, sendo utilizado em sistemas como MySQL, PostgreSQL, Oracle e SQL Server.

-- Conceitos Básicos de SQL --

Banco de Dados: conjunto organizado de informações.
Tabela: estrutura que armazena os dados em linhas e colunas.
Registro (linha): representa uma informação completa dentro da tabela.
Campo (coluna): representa um tipo de dado da tabela.
Chave Primária (Primary Key): identifica cada registro de forma única.
Consulta (Query): comando usado para buscar informações.

-- Principais Comandos SQL --

SELECT → consulta dados.
INSERT → adiciona novos dados.
UPDATE → altera dados existentes.
DELETE → remove dados.
CREATE TABLE → cria tabelas.
DROP TABLE → exclui tabelas.


Exemplo simples:

SELECT nome, idade
FROM alunos
WHERE idade >= 18;

Esse comando mostra o nome e a idade dos alunos maiores de 18 anos.
